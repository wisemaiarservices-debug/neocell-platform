# NeoCell North

## Target Environment

Canada, cold climates, remote sites, community infrastructure, and resilient microgrids.

## Purpose

Support monitoring of energy, storage, communications, weather exposure, and critical site operations.

## Connected Assets

- Solar arrays
- Batteries
- Backup generation
- Microgrid meters
- Community buildings
- Water systems
- Weather stations
- Communication gateways

## Demo Workflow

Storm risk forecast -> reduced solar forecast -> reserve battery planning -> critical load visibility -> operator recommendation.
